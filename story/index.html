資安學習心態 / Cybersecurity Mindset

你不需要最強的技術，
才能走下去

寫給每一個在資安學習路上，感到迷惘、痛苦、覺得自己不夠的人。

// 00 — 起點

從網咖長大的孩子，到踏進資安的世界

我從小在網咖長大。電腦對我來說，不是工具，是空氣。選擇資工系是自然的事，踏入資安，也是。

大學加入資安社團，第一次學 Linux 指令、第一次接觸 Web Security、第一次參加 CTF。那種第一次攻破某個東西的感覺——說不清楚比什麼都強，但你就是知道，這條路你想繼續走。

入門的感動，往往不來自你「懂了多少」，而是你第一次感覺「自己做到了」。

// 01 — 最深的傷

被否定的那一天

但路上有人告訴我：你沒有技術。

不是陌生人。是我敬重的學長。不是在私下。是在我出書的時候。

那種感覺不只是難過。是那種讓你開始懷疑——「我是不是真的不適合這裡？」、「我在這個領域，到底有沒有資格存在？」的那種難過。

我難過，而且我覺得⋯⋯他說的可能是對的。

如果你也曾經有過這種感受，我想讓你知道：你不孤單。資安圈有時候很殘酷，技術的高低很容易被拿來當作一個人「值不值得」的標準。但那不是全部的真相。

// 02 — 繼續走的理由

我為什麼沒有停下來

被否定之後，我沒有停。我繼續寫文章、繼續講課、繼續做 Lab、繼續實作。

不是因為我不在乎那些話。是因為我找到了一個比「證明自己」更大的理由。

01

寫文章，是寫給未來的自己

每一篇文章，都是在為某個還在迷路的人留下路標。那個人，有時候是別人，有時候就是幾年後的自己。

02

講課，是因為有人真的學到了

當學員說「謝謝你」、說「因為你的文章我學到了很多」、說「因為你讓我不再內疚」——那一刻，我知道我做的事情是有意義的。

03

做 Lab，是因為有人需要那個入口

資安的學習資源，對初學者往往太高牆。做一個 Lab，就是在那面牆上，開一扇讓人進來的門。

學長說我沒有技術。但我的學員說，因為我，他們踏進了這個領域。這兩件事同時是真的。而我選擇讓後者定義我繼續走下去的理由。

// 03 — 給你的話

如果你現在正在痛苦

資安學習會讓你痛苦，是正常的。這個領域太廣，永遠有人比你強，永遠有你不懂的東西，永遠有人用技術高低來評判你。

但我想告訴你一件事：

你不需要成為最強的人，才有資格在這條路上繼續走。你只需要，找到一個比被承認更重要的理由。

可能是某個讓你好奇的漏洞，可能是某個你想保護的系統，可能是某個你想幫助入門的人。找到那個理由，它會在你最動搖的時候撐著你。

我至今仍然想成為技術能力強的人。我沒有假裝那不重要。但我也不再讓「技術不夠強」成為停下來的藉口。

技術可以慢慢長，
但那個願意繼續走的心，
是你現在就已經擁有的東西。

— 一個從網咖長大、被否定過、但還在路上的資安學習者
